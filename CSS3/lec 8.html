What is CSS position?


Defines how elements are placed in the document.

Why is it important?


Controls layout, ensures responsiveness, and creates creative designs.


1:Static (default behavior)
2:Relative
3:Absolute
4:Fixed
5:Sticky



Static Position

Default position for all elements.
Follows the normal flow of the document.

Relative Position

Moves an element relative to its original position.
Does not affect the surrounding elements.

Absolute Position

Moves an element relative to the nearest positioned ancestor (or the viewport if none exists).
Removes the element from the normal document flow.

Fixed Position

Positions the element relative to the viewport.
Stays in place during scrolling.

Sticky Position

A hybrid of relative and fixed positioning.
Stays relative until a specified scroll position is reached, then becomes fixed.
